```

```
빠른 업데이트와 렌더링 속도

  * DOM이란, XML, HTML 문서의 각 항목을 계층으로 표현하여 생성, 변형, 삭제할 수 있도록 하는 인터페이스
  * DOM은 동기식, Virtual DOM은 비동기식 방법으로 렌더링
